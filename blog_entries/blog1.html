CodeQuest Devlog #1: Setting Up My Summer

Level 1: Planning, dreaming, and downloading Unity

5 min read

June 9, 2025

This week, I started my side quest of the summer.

I wanted to try something big and challenge myself by learning something new and amusing. That’s why I chose CodeQuest: The GameDev Adventure, a simulation game where you can chase the dream of becoming either a successful indie developer or part of a AAA studio, all within one year. The goal is to put myself in the shoes of an aspiring game developer and see what that journey could look like.

I’ve played games for as long as I can remember. I’ve always been drawn to simulation games, whether it’s The Sims, Bitlife, or even Minecraft. I love games where every choice leads to a different story. The future is unknown, and our choices are what define it.

This summer, I wanted to learn a new language and go beyond what I already know. I thought, why not hit two birds with one stone and learn how to build games at the same time? If I’m going to commit to a summer-long project, I want it to be something I truly care about, something that excites me.

CodeQuest is a simulation and adventure game built in 2D pixel art to bring out that retro vibe I’ve always loved. I’ll literally be playing as myself and living through the same challenges that my character will face in the game. That’s what makes this project different. I’m building it mainly for me. I want to understand what it feels like to live the daily life of a game developer and experience those challenges through this project.

I’ll be using C#, Unity, Figma, and a few other tools to bring this game to life. I plan to track my progress through weekly devlogs like this one, where I’ll share my experience, what I learn, and the bumps along the way. It’s not going to be easy. I’ll be juggling work, web development projects, school, and this game all at once. But I want the challenge. I want to push myself.

So far, I've

  • Installed all the tools I need
  • Found some potential pixel art assets
  • Written the Game Design Document
  • Started the project README

Next Week's Plan:

  • Learn basic C# so I can start scripting in Unity
  • Design the UI layout in Figma
  • Build a simple Unity scene with placeholder stats and action buttons
